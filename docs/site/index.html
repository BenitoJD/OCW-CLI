OCW CLI

OpenCode Go workers for Codex-style orchestration. Keep your premium agent as orchestrator and delegate bounded exploration, review, PR summaries, and patch drafts to cheaper worker models.

GitHub Releases Codex Setup

Install

brew install BenitoJD/ocw/ocw
curl -fsSL https://raw.githubusercontent.com/BenitoJD/OCW-CLI/main/scripts/install-release.sh | bash

Core Flow

ocw explore "Map the auth flow"
ocw review "Review the current diff"
ocw --worktree patch "Draft the smallest safe fix"
ocw apply latest --check

Agent Integrations

Codex, Claude Code, OpenCode, and MCP setup snippets are built in.

PR Review

Use ocw pr summary and ocw pr review to create local artifacts without posting comments.

Safety

Audit, manifest, support bundle, policy checks, isolated worktrees, and release attestations.

Observability

Trace, report, stats, eval, and deterministic gauntlet tests for confidence.
